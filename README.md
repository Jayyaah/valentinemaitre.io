**Lien :** https://jayyaah.github.io/valentinemaitre.io/

---

## Contenu

- **Accueil** — présentation et liens rapides
- **Parcours** — expériences professionnelles et formations avec détails en modal
- **Projets** — galerie filtrée par technologie (iOS, Web, Android, Java)
- **Contact** — formulaire via Formspree

## Stack

- HTML5 / CSS3 / JavaScript
- Bootstrap 5.3
- Bootstrap Icons
- Formspree (formulaire de contact)
