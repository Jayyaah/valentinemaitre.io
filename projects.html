Passe P'Art Où !

Description :
Passe P'Art Où ! est un prototype d’application iOS axé sur la découverte artistique, inspiré du concept de Pokémon Go.

📱 L’application permet aux utilisateurs de :

  • Explorer une carte interactive avec différents lieux d’art (théâtres, musées, galeries, etc.)
  • Réaliser des missions liées à ces lieux pour débloquer des récompenses
  • Échanger ces récompenses dans une boutique virtuelle contre divers items
  • Personnaliser leur interface utilisateur avec les items gagnés

🚀 Fonctionnalités principales :

  • Carte interactive : visualisation des lieux d’art autour de soi
  • Missions : défis à accomplir dans différents endroits culturels
  • Récompenses : système de gains et échanges dans la boutique intégrée
  • Personnalisation : customisation de l’interface grâce aux items obtenus

🧑‍💻 Stack technique :

  • Swift / SwiftUI
  • MapKit pour la carte interactive
  • Architecture MVVM (modèle-vue-vue modèle)
  • Gestion des données locales

EasyTravel

Description :
EasyTravel est un prototype d'application iOS permettant aux utilisateurs de découvrir des destinations adaptées à leurs préférences.

🛠️ Fonctionnement :

  • Quiz de préférences pour filtrer les destinations
  • Fiches complètes : prix, images, descriptions
  • Vols : horaires, compagnies, prix, liens de réservation
  • Hébergements : nom, adresse, prix, liens de réservation
  • Indicateur de pertinence des suggestions

🧑‍💻 Stack technique :

  • Swift
  • SwiftUI
  • Xcode

GamePicker

Description :
GamePicker est un site web simple et fun pour gérer ta bibliothèque de jeux vidéo et choisir quoi jouer facilement.

📝 Ce que fait l’app :

  • Ajoute des jeux à ta bibliothèque personnelle
  • Filtre par plateforme : PC, PlayStation, Xbox, Switch...
  • Tire un jeu aléatoire selon toutes ou une plateforme spécifique
  • Aide à choisir un jeu rapidement quand tu hésites

🚀 Fonctionnalités principales :

  • Gestion de bibliothèque : ajouter, modifier, supprimer
  • Filtrage dynamique par plateforme
  • Tirage au sort d’un jeu au hasard
  • Stockage des données avec une base MySQL
  • Interface simple, claire, fluide en HTML/CSS

🧑‍💻 Stack technique :

  • PHP (logique serveur)
  • MySQL pour la base de données
  • HTML & CSS pour l’interface

💡 Idées d’améliorations futures :

  • Comptes utilisateurs
  • Ajout d’un système de notes/évaluations
  • Suggestions de jeux selon préférences/historique
CMS from Scratch

Description :
Prototype de CMS développé de zéro, sans framework, reprenant les bases d’un système comme WordPress.

📝 Ce que fait le CMS :

  • Gestion des utilisateurs avec rôles et validation par email
  • CRUD complet sur les pages, articles et commentaires modérés
  • Support du sitemap et des optimisations SEO
  • Thèmes multiples et système de dashboard admin

🧑‍💻 Stack technique :

  • PHP 7 (logique serveur)
  • HTML5 / CSS3 / SCSS pour le front
  • jQuery pour les interactions dynamiques
  • PhpMailer pour l’envoi d’emails
  • Datatables pour les tableaux dynamiques

🗂️ Méthodologie :

  • Versionné avec GitHub
  • Organisation des tâches sous Trello
  • Architecture MVC simplifiée (from scratch)
Mario Kart IRL

Description :
Application Android développée pour piloter un robot Arduino à distance via un serveur intermédiaire.

🎮 Fonctionnalités principales :

  • Contrôle par gyroscope du smartphone
  • Communication en temps réel via sockets TCP/IP

🧑‍💻 Stack technique :

  • Java (application Android)

  • Sockets TCP/IP pour la communication
  • Capteurs gyroscopiques du téléphone
